# GitHub网站汉化
不是内容，是功能文字的汉化
1. 安装用户脚本管理器：
    - 推荐：[Tampermonkey](http://tampermonkey.net/ "篡改猴")
![](assets/GitHub网站使用技能/file-20251211092002491%201.png)
2. **Chrome / Chromium 内核浏览器，请务必开启 “扩展程序” 管理中的 “开发者模式”[1](https://github.com/maboloshi/github-chinese?tab=readme-ov-file#user-content-fn-1-09aac711790e8e57fc720af71d603ee7)**
3. 选择安装源：
    - [GitHub 源【开发版】](https://github.com/maboloshi/github-chinese/raw/gh-pages/main.user.js "GitHub 中文化插件 - GitHub 源")
    - [GreasyFork 源【稳定版】](https://greasyfork.org/scripts/435208-github-%E4%B8%AD%E6%96%87%E5%8C%96%E6%8F%92%E4%BB%B6/code/GitHub%20%E4%B8%AD%E6%96%87%E5%8C%96%E6%8F%92%E4%BB%B6.user.js "GitHub 中文化插件 - GreasyFork 源")
    - 点击后有可能会下载一个JS文件GitHub_中文化插件.user.js，将他拖入到篡改猴的网页中。
![](assets/GitHub网站使用技能/file-20251211092002491.png)
4. 刷新页面后，插件即可生效
5. 必要时，重启浏览器
