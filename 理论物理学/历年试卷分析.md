### 3. 试题解答

#### 一、定义与微分表示
1.  **亥姆霍兹自由能（Helmholtz free energy, $F$）**
    - 定义：$F = U - TS$，其中$U$为内能，$T$为温度，$S$为熵。
    - 自然变量：$T, V, N$
    - 微分形式：$dF = -SdT - PdV + \mu dN$
    （物理意义：等温等容下，系统对外做功的最大值；平衡态下，$F$取极小值）

2.  **吉布斯自由能（Gibbs free energy, $G$）**
    - 定义：$G = H - TS = U + PV - TS$，其中$H$为焓，$P$为压强，$V$为体积。
    - 自然变量：$T, P, N$
    - 微分形式：$dG = -SdT + VdP + \mu dN$
    （物理意义：等温等压下，系统对外非体积功的最大值；平衡态下，$G$取极小值）

3.  **巨势（grand potential, $\Omega$）**
    - 定义：$\Omega = F - \mu N = U - TS - \mu N$，其中$\mu$为化学势，$N$为粒子数。
    - 自然变量：$T, V, \mu$
    - 微分形式：$d\Omega = -SdT - PdV - Nd\mu$
    （物理意义：等温等容等化学势下，系统平衡态时$\Omega$取极小值，且平衡时$\Omega = -PV$）

#### 二、与配分函数的关系
1.  **亥姆霍兹自由能与正则系综**
    正则系综（$T, V, N$固定）的配分函数为：
    $$Z = \sum_i e^{-\beta E_i}, \quad \beta = \frac{1}{k_B T}$$
    亥姆霍兹自由能与配分函数的关系为：
    $$F = -k_B T \ln Z$$

2.  **吉布斯自由能与正则系综（或压强正则系综）**
    吉布斯自由能可通过正则系综的亥姆霍兹自由能推导：$G = F + PV$，其中$P = -\left(\frac{\partial F}{\partial V}\right)_{T,N}$。
    也可通过压强正则系综（$T, P, N$固定）的配分函数$\Delta$直接表示：
    $$G = -k_B T \ln \Delta, \quad \Delta = \int_0^\infty Z(V) e^{-\beta PV} dV$$

3.  **巨势与巨正则系综**
    巨正则系综（$T, V, \mu$固定）的配分函数为：
    $$\Xi = \sum_{N=0}^\infty \sum_{i(N)} e^{-\beta (E_i - \mu N)} = \sum_{N=0}^\infty Z(N) e^{\beta \mu N}$$
    巨势与巨配分函数的关系为：
    $$\Omega = -k_B T \ln \Xi$$
    结合平衡态时$\Omega = -PV$，可得：$PV = k_B T \ln \Xi$。

#### 三、统计物理问题求解的一般步骤
1.  **确定系统与约束条件**：明确系统的组成（粒子种类、相互作用）、约束条件（如固定$T,V,N$/$T,P,N$/$T,V,\mu$），选择合适的系综（正则/巨正则/微正则）。
2.  **写出系统的哈密顿量**：根据系统的微观模型（如理想气体、晶格振动、自旋系统），写出单粒子或系统的哈密顿量$H$。
3.  **计算配分函数**：根据所选系综，计算对应的配分函数（正则配分函数$Z$或巨配分函数$\Xi$），对连续系统需将求和转化为相空间积分（考虑量子简并时引入修正因子）。
4.  **通过热力学势导出宏观量**：利用配分函数与热力学势的关系（如$F=-k_B T\ln Z$，$\Omega=-k_B T\ln\Xi$），通过热力学势的偏导数，求解压强$P$、熵$S$、内能$U$、粒子数$N$、化学势$\mu$等宏观热力学量。
5.  **分析结果的物理意义**：对得到的宏观量进行分析，验证其与热力学定律的一致性，或解释系统的热力学行为（如相变、状态方程）。
