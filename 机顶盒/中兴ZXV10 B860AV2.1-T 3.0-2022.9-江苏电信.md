# 一、前提与风险说明（必读）

- 机型适配：中兴ZXV10 B860AV2.1-T 3.0-2022.9（江苏电信），**仅S905L3/B芯片、非高安版**可刷安卓9.0+；高安版（带CA加密）几乎无法刷入第三方安卓9，强行刷入会变砖。
- 风险：刷机将**失去运营商保修、清除所有数据、可能导致WiFi/网口/遥控异常**，操作失误有砖机风险，后果自负。
- 固件来源：优先ZNDS论坛、宽带技术网等社区的**S905L3B专用安卓9线刷包**（.img格式），如B860AV2.1-T 3.0_S905L3B_uw5621无线版（适配江苏电信）。

### 二、刷机前准备
#### 硬件工具
1. 双公头USB 2.0线：长度30–50cm，优先带磁环，插机顶盒**靠近网口的USB口**。
2. 短接工具：
   - 免拆短接神器（晶晨专用，插入AV/HDMI口模拟短接，推荐）。
   - 无神器则拆机，用镊子短接主板**C9AB**/C122/C128/R89（B860AV2.1-T 3.0通用短接点）。
![](assets/中兴ZXV10%20B860AV2.1-T%203.0-2022.9-江苏电信/短接点C9AB.jpg)
3. Windows电脑（Win7/10/11，关闭杀毒/防火墙、禁用驱动签名）。
4. 电源适配器、HDMI线、电视（用于后续验证）。

#### 软件与固件
1. 刷机工具：**Amlogic USB Burning Tool v2.2.3/2.2.4**（晶晨芯片专用，低版本易报错）。
2. 驱动：安装工具自带的Amlogic USB驱动；Win10/11需**关闭驱动签名强制验证**后安装。
3. 固件包：下载**B860AV2.1-T 3.0_S905L3B_安卓9.0线刷包**（.img，解压后使用）。
   - 参考路径：ZNDS→当贝纯净桌面→固件大全→中兴盒子B860合集→B860AV2.1-T 3.0。

### 三、详细刷机步骤（线刷）
#### 步骤1：电脑端配置
1. 安装Amlogic USB Burning Tool，以**管理员身份**运行。
2. 点击【文件】→【导入烧录包】，选择解压后的安卓9.0 .img固件。
3. 烧录配置：**取消勾选“擦除flash”“擦除bootloader”**（新手必设，避免砖机）；若识别失败再勾选重试。
4. 点击【开始】，工具进入等待设备连接状态。

#### 步骤2：机顶盒短接+连接（关键）
1. 机顶盒**断电**，双公头USB一端插电脑USB 2.0口，另一端插机顶盒**靠近网口的USB口**。
2. 短接操作（二选一）：
   - 免拆：插入短接神器到AV/HDMI口，保持插入状态。
   - 拆机：用镊子短接主板C122/C128/R89，保持短接不松手。
3. 保持短接，**接通机顶盒电源**；电脑识别设备后（工具提示“连接成功”），立即松开短接/拔出神器。

#### 步骤3：开始刷机
1. 工具自动开始烧录，进度条从0%→100%；期间**不要断电、不要拔USB**。
2. 常见问题：
   - 进度7%报错：关闭提示，若设备仍连接，重新点击【开始】；失败则断电重短接重试。
   - 98%失败：更换Amlogic USB Burning Tool v2.2.4版本，或换固件包。
3. 进度100%、提示“烧录成功”后：
   - 先点击工具【停止】→【关闭】。
   - 断电机顶盒，拔掉USB线与短接工具。

#### 步骤4：首次开机与验证
1. 机顶盒接HDMI、电源，通电开机；**首次启动较慢（1–3分钟）**，耐心等待初始化。
2. 进入安卓9.0桌面（如当贝桌面），验证：
   - 系统设置→关于：确认安卓版本9.0、内核匹配S905L3B。
   - 网络：有线/WiFi（uw5621）是否正常，江苏电信网络需在设置中清空认证、设DHCP鉴权为noauth。
   - 遥控：若原厂遥控失效，安装当贝遥控/电视家等APP适配。

### 四、救砖方案（刷机失败/变砖）
1. 断电重短接：重新执行“短接+通电+USB连接”，换固件/工具版本重试。
2. TTL救砖：用CH340串口模块连接主板TTL引脚，电脑端用SecureCRT输入命令强刷（适合高手）。
3. 回退原厂：若安卓9不稳定，下载B860AV2.1-T 3.0原厂线刷包，按同样步骤刷回。
