要让Obsidian新建文件默认存到**指定文件夹、且支持多个文件夹**，分「官方基础设置（单默认）」和「插件方案（多文件夹/智能选择）」两步，下面是完整可直接操作的方法：

### 一、基础设置：固定一个默认文件夹（官方自带，无需插件）
1. 打开Obsidian → 左下角**设置（齿轮）** → 进入**文件与链接**
2. 找到「**新建笔记的存放位置**」，下拉选：
   - 「**在指定文件夹中**」→ 输入目标文件夹路径（如 `Inbox`、`日记/2026`、`工作/项目A`），直接写文件夹名即可（已存在的文件夹）
   - 可选「**与当前文件相同的文件夹**」：打开哪个文件夹里的笔记，新建就放那里，适合多文件夹场景的基础用法
3. 保存设置，之后按 `Ctrl/Cmd+N` 新建，就会自动进你指定的文件夹，不再在根目录

### 二、多文件夹方案：一键新建到不同指定文件夹（插件实现）
#### 方案1：Advanced New File（最常用，快速选文件夹）
1. 第三方插件市场搜索安装 **Advanced New File**，启用
2. 设置：打开插件设置，可预设常用文件夹列表、自定义快捷键
3. 使用：按 `Ctrl/Cmd+P` 打开命令面板，搜「Advanced New File」→ 选择目标文件夹 → 输入文件名，直接在该文件夹新建；也可设专属快捷键（如 `Ctrl+Alt+N`），一键调出文件夹选择

#### 方案2：Create Note In Folder + QuickAdd（多预设文件夹，一键直达）
1. 安装 **Create Note In Folder** + **QuickAdd**
2. Create Note In Folder设置：添加多个「文件夹+命令名」（如：新建到Inbox、新建到日记、新建到工作），每个生成独立命令
3. QuickAdd设置：把这些命令做成按钮/快捷入口，或绑定不同快捷键（如 `Ctrl+Shift+1` 到Inbox、`Ctrl+Shift+2` 到日记）
4. 直接按对应快捷键，新建文件就自动进入目标文件夹

#### 方案3：Templater + 模板（按类型自动归到对应文件夹）
适合按笔记类型（日记、工作、读书）自动归类：
1. 安装 **Templater**，启用
2. 新建模板（如「日记模板」），在模板顶部加：
```
<%*
tp.file.move("/日记/" + tp.file.title)
%>
```
3. 用该模板新建笔记，会自动移动到「日记」文件夹；同理做「工作」「读书」模板，分别对应不同文件夹

### 三、手动快速新建到指定文件夹（无需插件）
- 右键左侧文件列表里的目标文件夹 → 选「**新建笔记**」，直接在该文件夹创建
- 拖拽新建文件：先在根目录新建，再拖到目标文件夹（开启「设置→文件与链接→自动更新内部链接」，避免链接失效）

### 四、自动归档（新建后自动移到对应文件夹）
安装 **Auto Note Mover** 插件，设置规则：
- 监控根目录新建文件
- 按文件名关键词/标签/模板，自动移动到预设文件夹（如含「日记」→ 日记文件夹，含「工作」→ 工作文件夹）
