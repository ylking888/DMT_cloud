根据视频内容，各大 vibe coding 工具的排名可按 “上手门槛、智能程度、性价比” 综合分为以下梯队（从高到低）：

## 第一梯队：顶级工具
### Cloud Code（CC）
优势：架构强大，支持接入国内大模型（如 GLM、Minimax），适合硬核开发者。
劣势：上手门槛极高（需终端操作 + 海外模型限制），对新手不友好。
排名：穷中之穷（仅限硬核用户）。

### Codex（OpenAI）
优势：国区友好，无需终端操作，集成 GPT 模型，适合新手和科研场景。
劣势：仅支持自家模型，高级功能需 200 刀 / 月套餐。
排名：顶级（综合体验最佳）。

### Antigravity（谷歌）
优势：极速响应 + 极简设计，支持 Gemini 模型，适合快速修改代码。
劣势：谷歌限制用量，模型能力有 “降智” 趋势。
排名：人上人（曾顶级，现略降）。

## 第二梯队：中高端工具
### Cursor
优势：第三方 IDE，支持接入御三家模型（GPT/Claude/Gemini）。
劣势：依赖外部模型，常被限速或限制（如 Claude），定位尴尬。
排名：顶级（适合多模型需求，但非首选）。

### WindSurf
优势：上下文管理优秀，适合企业级代码修改（如 “屎山代码”）。
劣势：曾与 OpenAI 发生纠纷，稳定性存疑。
排名：人上人（功能强，但需谨慎使用）。


### Trae（字节跳动）
优势：国内工具，支持 GLM、Kimi 等主流模型，上手门槛低。
劣势：整体能力略逊于海外工具。
排名：人上人（国内首选）。

## 第三梯队：入门级工具
### Qoder（阿里）/ CodeBuddy（腾讯）
优势：支持国内模型，基础功能可用。
劣势：上下文管理、工程能力较弱，Agent 功能差一档。
排名：NPC（普通辅助工具）。


### Replit / Lovable
优势：0 代码门槛，通过自然语言生成 App，适合小白。
劣势：代码黑盒，定制性低。
排名：
Replit：Hunk（更美观、周全）。
Lovable：顶级（适合快速原型）。

## 总结推荐
- 0 基础小白：Replit（首选）、Lovable（次选）。
- 有经验开发者：Cursor（多模型尝试）、WindSurf（企业级需求）。
- 硬核开发者：Cloud Code（首选）、Codex（辅助）。
- 国内用户：Trae（首选）、Qoder/CodeBuddy（备用）。
