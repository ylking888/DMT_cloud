#### 安装绘图插件Excalidraw
	
![](assets/2026-04-13/file-20260413171218321.png)
	设置Excalidraw插件里的两个快捷键
		ALT+E：Excalidraw: 在 Excalidraw 和 Markdown 模式之间切换
		CTRL+ALT+E：Excalidraw: 转换：空白 Markdown 文档 => Excalidraw 绘图
	可以在Excalidraw中插入“背景笔记”卡片，就可以编辑md文件内容同步，如果插入最大的总标题，就可把全部md插入到Excalidraw中了![](assets/2026-04-13/file-20260413172320371.png)
	插入这段笔记之后，我还可以把边框变成透明的；还可以在旁边进行绘画。这样我们就实现了文本和图片同框操作。

#### 自定义的CSS
```CSS
/* 通用Callout竖条基础样式 */
.callout {
  background-color: transparent !important;
  border: none !important;
  border-left: 8px solid !important; /*竖条粗细*/
  padding: 4px 24px !important; /*竖条高度、文字与竖条距离*/
  margin: 4px 0 !important; /*竖条间距*/
  border-radius: 0px !important; /*给竖条加圆角*/
}
.callout .callout-icon {
  display: none !important;
}
.callout .callout-title {
  color: #1e1e1e !important;
  font-size: 1.5em !important;
  font-weight: bold !important;
}

/* 蓝色（info/note）：知识/笔记类 */
.callout[data-callout="lanse"]{
  border-left-color: #4a90e2 !important;
}

/* 绿色（success）：完成/进度类 */
.callout[data-callout="lvse"] {
  border-left-color: #4caf50 !important;
}

/* 黄色（warning）：提醒/待办类（你最初的样式） */
.callout[data-callout="huangse"] {
  border-left-color: #ffcc00 !important;
}

/* 橙色（question）：疑问/思考类 */
.callout[data-callout="chengse"] {
  border-left-color: #ff9800 !important;
}

/* 红色（danger/failure）：风险/错误类 */
.callout[data-callout="hongse"],
.callout[data-callout="红色"] {
  border-left-color: #f44336 !important;
}

/* 紫色（example/bug）：示例/问题类（可选） */
.callout[data-callout="zise"] {
  border-left-color: #9c27b0 !important;
}

/* 适配深色主题：文字自动变白 */
.theme-dark .callout .callout-title {
  color: #ffffff !important;
}

```
> [!lanse] 蓝色提醒色块

> [!lvse] 绿色提醒块
> 测试内容

自定义竖条宽度 / 圆角
修改 CSS 里的 border-left: 8px solid !important;，
比如改成 4px 变细，12px 变粗；
新增 border-radius: 4px !important; 给竖条加圆角

# 显示大纲 {.lanse}
在“核心插件”——“大纲”中打开，右侧就会多一个大纲视图
鼠标按住大纲面板的标题栏（Outline），拖到左侧边栏（和文件列表、搜索并排）
或点击右侧边栏大纲面板右上角的三点菜单 → 移动到左侧边栏
你可以设置一个快捷键来快速打开大纲，一般使用Ctrl+Shift+O
